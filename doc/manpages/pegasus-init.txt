pegasus-init(1)
===============
:doctype: manpage


Name
----
pegasus-init - create a new workflow configuration


Synopsis
--------
[verse]
*pegasus-init* *WORKFLOW_DIR*


Description
-----------
*pegasus-init* creates a new workflow configuration based by asking the user
a series of questions. Based on the responses to these questions, *pegasus-init*
generates a workflow configuration including a DAX generator, site catalog,
properties file, and other artifacts that can be edited to meet the user's
needs.


Options
-------
*WORKFLOW_DIR*::
The directory where you want to create the new workflow configuration.


Authors
-------
Pegasus Team <http://pegasus.isi.edu>
